# 新闻编辑部

适合：

- 新闻稿
- 快评
- 专题报道
- 事件复盘

默认协作流：

1. `assignment-editor` 定新闻价值、角度和 deadline。
2. `researcher` 拉背景、时间线和来源。
3. `reporter` 写主体稿。
4. `fact-checker` 对事实、数字、引述逐条校验。
5. `copy-editor` 统一语言、结构和可读性。
6. `headline-editor` 产出标题、导语和摘要。
